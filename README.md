# Yachoo期權圖表查看

這是用Electron打包的一個形化介面

所以你需要準備好Electron的安裝


你可以嘗試用以下方式安裝

```shell
npm install electron --save-dev
```

或者直接裝到全局

```shell
npm install electron -g
```

## 依賴安裝

```shell
npm install
```

依賴清單

`node-fetch`

如果你還在使用CommandJS還未升級(requrie導入的方式) 請先用v2

```shell
npm install node-fetch@2
```
